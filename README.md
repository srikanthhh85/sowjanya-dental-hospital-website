# 🦷 Sowjanya Dental Hospital – Responsive Healthcare Website

A modern, responsive multi-page healthcare website developed for **Sowjanya Dental Hospital**. The project focuses on delivering an engaging user experience with appointment booking, service information, patient testimonials, blogs, and WhatsApp consultation integration.

---

## 📌 Overview

This project was developed as a client-focused healthcare website to improve the hospital's online presence and simplify appointment booking for patients.

The website provides an intuitive interface where visitors can explore dental services, read educational blogs, locate the hospital, and book appointments directly through WhatsApp.

---

## ✨ Features

- 🦷 Responsive Landing Page
- 📱 Mobile-Friendly Navigation
- 📅 Appointment Booking Form
- 💬 WhatsApp Appointment Integration
- 📍 Google Maps Integration
